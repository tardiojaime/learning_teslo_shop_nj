## WebSockets

```bash
yarn add @nestjs/websockets @nestjs/platform-socket.io
```

## Habilitar CORS - main.ts

```bash
app.enableCors();
app.enableCors( options );
@WebSocketGetaway({ cors: true });
# Probar conexion 
localhost:3000/socket.io/socket.io.js
```
